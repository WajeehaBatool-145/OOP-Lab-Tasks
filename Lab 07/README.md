# 🧮📊 Calculator Systems (Composition + Aggregation)

C++ programs demonstrating **composition (Calculator–Display ownership)** and **aggregation (shared Calculator used by multiple Students)** for performing and displaying arithmetic operations.


## 🧮 Calculator with Display System

A C++ program demonstrating **composition**, where a `Calculator` class uses a separate `Display` class to show results and store the last calculated value. 📊✨

## 🧑‍🎓 Shared Calculator System

A C++ program demonstrating **aggregation**, where multiple `Student` objects use a single externally managed `Calculator` to perform addition and multiplication operations. ➕✖️📊
