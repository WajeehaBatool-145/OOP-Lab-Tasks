# All four programs demonstrate **C++ inheritance with constructor/destructor chaining and memory management (including polymorphism and virtual destructors)** in different real-life examples.


## 📺 Electronics Product Management System

A simple C++ inheritance program demonstrating constructor chaining between `Product` and `Electronics` classes with product details and warranty information display. ✨

## 🚗 Electric Vehicle Constructor Chaining

A C++ multilevel inheritance program demonstrating constructor chaining from `Vehicle` to `Car` and `ElectricCar` using parameterized constructors. ⚡

## 📄 File Handling with Destructors

A basic C++ inheritance program showing constructor and destructor execution order using `File` and `TextFile` classes. 🔁

## 🌡️ Smart Sensor Device Management

A C++ inheritance program demonstrating dynamic memory allocation, virtual destructors, and heap memory management using `Device` and `Sensor` classes. 🔧
