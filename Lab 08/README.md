# 💼📊 OOP Polymorphism Programs

C++ programs demonstrating **runtime polymorphism** using shape area calculation and employee salary system with function overriding via base class pointers.


## 📐 Shape Area Calculator

A C++ program demonstrating **runtime polymorphism**, where a base class `Shape` pointer calls overridden `area()` functions of `Rectangle` and `Circle` to calculate different areas. 🔺⭕📊

## 💼 Employee Salary System

A C++ program demonstrating **runtime polymorphism**, where a base class `Employee` pointer calculates salaries differently for `PermanentEmployee` and `ContractEmployee` using function overriding. 💰📊
