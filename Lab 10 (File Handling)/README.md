# 📁 File Handling Programs in C++

A set of C++ programs demonstrating **file operations (write, read, append, copy, and line counting)** using `fstream` to manage and process text files. 📄🔄📊

## 📁 File Handling in C++

A C++ program demonstrating **file creation, writing, reading, and appending** using `fstream` to manage data in `notes.txt`. 📝📂

## 📄 File Line Counter in C++

A C++ program that reads a file (`notes.txt`) and counts the total number of lines using file handling and `getline()` function. 🔢📁

## 📂 File Copy Program in C++

A C++ program that demonstrates **file handling**, where content is read from `source.txt` and copied line by line into `destination.txt` using `getline()` and `fstream`. 🔄📄

## 🎓 Student File Handling Program in C++

A C++ program that **writes student names and roll numbers to a file (`students.txt`) and then reads and displays all records using file handling and `getline()`**. 📄📚
