# 🟢 1. Static Data Members
## 🧠 Concept:

### Static data members are shared variables among all objects of a class. Only one copy exists in memory.

## 📌 Key Points:
### Declared using static keyword

### Shared by all objects
### Accessed using class name
## 💡 Use Case:

### Used for counting objects or storing common data like company name.

# 🔵 2. Abstract Class
## 🧠 Concept:

### An abstract class is a class that cannot be instantiated and contains at least one pure virtual function.

## 📌 Key Points:
### Uses virtual function = 0

### Must be inherited

### Forces derived classes to implement functions
## 💡 Use Case:

### Used in real-world modeling like:


### Hospital system (Doctor, Nurse, Receptionist)

# 🟣 3. Friend Function
## 🧠 Concept:

### A friend function is a non-member function that can access private and protected members of a class.

## 📌 Key Points:
### Declared using friend keyword
### Not part of class but has access
### Used for special operations
## 💡 Use Case:

### Useful for operator overloading or accessing private data safely.

## 🎯 Conclusion

### These concepts are essential for understanding advanced OOP design in C++:

### Static → shared data handling
### Abstract → structure & abstraction
### Friend → controlled access to private data
