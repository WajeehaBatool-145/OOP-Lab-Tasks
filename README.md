# Object-Oriented Programming (OOP) 

## 📌 Overview
This repository contains my practice work and assignments related to Object-Oriented Programming (OOP) in C++. It covers fundamental to intermediate concepts used in software development and real-world system design.

The main goal of this work is to understand how OOP helps in building modular, reusable, and maintainable code.

---

## 🧠 Concepts Covered

### 1. Classes and Objects
- Learned how to define classes in C++
- Created objects to access class members
- Understood real-world modeling using classes

---

### 2. Encapsulation
- Used private and public access specifiers
- Protected data from direct external access
- Implemented getter and setter functions

---

### 3. Constructors and Destructors
- Default and parameterized constructors
- Constructor overloading
- Destructor for memory cleanup
- Dynamic memory management using `new` and `delete`

---

### 4. Inheritance
- Single and hierarchical inheritance
- Reusability of code using base and derived classes
- Extended functionality in child classes

---

### 5. Polymorphism
#### Compile-Time Polymorphism
- Function overloading
- Operator overloading

#### Run-Time Polymorphism
- Function overriding
- Virtual functions
- Base class pointers pointing to derived objects

---

### 6. Virtual Functions
- Achieved dynamic binding
- Used for runtime decision making
- Implemented base class interface behavior

---

### 7. Abstraction
- Hiding implementation details
- Using abstract concepts to focus on essential features
- Use of base classes to define structure

---

### 8. Operator Overloading
- Customized operators for user-defined classes
- Improved readability and natural syntax for objects

---

### 9. Dynamic Memory Allocation
- Allocation using `new`
- Deallocation using `delete`
- Preventing memory leaks using destructors

---

### 10. Pointers in OOP
- Object pointers
- Base class pointers to derived objects
- Understanding memory-level object handling

---

### 11. Real-world Applications
- Employee Salary System (Permanent & Contract Employees)
- Device Management System with inheritance
- Calculator system using class interaction
- Shape-based area calculation system
