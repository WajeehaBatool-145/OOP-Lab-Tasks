## 📚 These programs demonstrate Object-Oriented Programming (OOP) in C++ using classes, structures, header files, and member functions to handle real-world data and operations.


**👨‍🎓 Student Data Input and Display using Header File and Class in C++ 📊** — A program that takes student details from the user, stores
them using a class defined in a header file, and displays the information.

**📐Rectangle Class in C++ using Header File (OOP Implementation) 📊** — A program that defines a Rectangle class in a header file to take dimensions and calculate area and perimeter.

**🗳️ Voter Eligibility Check using Class and Header File in C++ 📊** — A program that takes user details and determines voting eligibility based on age using a class function.

**🌡️ Temperature Conversion using Class in C++ 📊** — A simple program that converts temperature from Celsius to Fahrenheit using a class and displays the result.

**🧮 Basic Calculator using Class and Header File in C++ ➕➖✖️➗** — A program that performs basic arithmetic operations (addition, subtraction, multiplication, and division) using class member functions defined outside the class.
